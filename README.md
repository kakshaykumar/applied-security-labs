# applied-security-labs

A mini portfolio of applied security exercises:

```text
applied-security-labs/
├── README.md
├── 01-sql-injection/
│   ├── README.md
│   └── injection-attack.sql
└── 02-rsa-cryptography/
    ├── README.md
    ├── public.pem
    ├── private.pem
    └── data.txt.enc
```

- **01-sql-injection**: Demonstrates SQL injection attack mechanics and secure query mitigation.
- **02-rsa-cryptography**: Demonstrates RSA key generation, encryption/decryption flow, and key-management context.
